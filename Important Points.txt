Important Commands and Points 

1) Commands for creating environment in a anaconda via terminal 

conda create -p venv python==3.12.7 -y

2) Git Commands : 
a) git init --- Initializing the git 
b)git add README.md --- To add the readme file 
c) git commit -m "Message Here " ---- To commit the changes 
d) git branch -M main --- Change the branch name 
e) git remote add origin "github project url" --- Connect git to github for project synchronization
f)git push -u origin main ---- Push all the changes to github
